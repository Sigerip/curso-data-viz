3. Matplotlib#

MATPLOLIB é uma biblioteca de software para criação de gráficos e visualizações de dados em geral, feita para e da linguagem de programação Python e sua extensão de matemática NumPy.

SciPy, biblioteca voltada para computação científica e técnica, oferece uma vasta coleção de algoritmos matemáticos e funções para diversas áreas, como álgebra linear, otimização, integração numérica, estatística e processamento de sinais, utiliza o Matplotlib para criar visualizações gráficas dos dados e resultados.

Originalmente criada pelo biólogo e neurocientista americano John D. Hunter, a biblioteca hoje possui uma comunidade ativa atuando em seu desenvolvimento e é distibuída sob uma licença BSD. O programador Michael Droetboom foi nomeado o líder do projeto um pouco antes da morte do criador John Hunter em agosto de 2012, e logo o cientista Thomas Caswell se juntou a ele.

Oferece uma interface de programação orientada a objetos para incluir gráficos em aplicações usando toolkits de interface gráfica, como Tkinter, WxPython, Qt ou GTK.

Matplotlib 2.0.x suporta Python nas versões 2.7 até 3.6. O suporte ao Python 3 começou com o Matplotlib 1.2, e o Matplotlib 1.4 foi o último a suportar o Python 2.6. O projeto se comprometeu a não dar mais suporte ao Python 2 a partir de 2020, assinando o Python 3 statement.

3.1 Anatomia de um Gráfico Matplotlib#

Antes de começarmos a criar gráficos, é importante entender a estrutura de um gráfico Matplotlib:

  • Figure: O contêiner de nível superior que pode conter múltiplos eixos (plots)

  • Axes: O espaço onde os dados são plotados

  • Axis: As linhas que delimitam os limites do gráfico (eixos x e y)

  • Artist: Tudo o que você vê no gráfico (linhas, textos, legendas, etc.)

# Bibliotecas
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy import stats

No código abaixo, ax é um objeto que representa os eixos do gráfico (axes em inglês) e funciona como uma “tela de pintura” onde todos os elementos do gráfico são desenhados. Quando usamos plt.subplots(), ele retorna dois objetos: fig (a figura completa, como se fosse a moldura) e ax (a área onde efetivamente desenhamos o gráfico). O ax é particularmente importante porque é através dele que controlamos praticamente tudo no gráfico.

Esta abordagem usando ax é considerada mais “orientada a objetos” e oferece maior controle sobre os elementos individuais do gráfico, sendo especialmente útil quando precisamos criar gráficos mais complexos ou múltiplos subplots na mesma figura.

# Criar uma figura, um conjunto de eixos e define o tamanho do gráfico em polegadas (largura x altura)
fig, ax = plt.subplots(figsize=(10, 6))

# Gerar dados
x = np.linspace(0, 10, 100) # cria um array com 100 números, distribuídos uniformemente entre 0 e 10 (inclusive)
y = np.sin(x) # calcula o seno de cada valor de x

# Plotar dados, desenhar a linha do gráfico e define a legenda
ax.plot(x, y, label='sen(x)')

# Adicionar elementos ao gráfico
ax.set_title('Função Seno', fontsize=15) # define o título
ax.set_xlabel('x', fontsize=12) # define o rótulo do eixo x e o tamanho da fonte
ax.set_ylabel('sen(x)', fontsize=12) # define o rótulo do eixo y e o tamanho da fonte
ax.legend(fontsize=12) # adiciona a legenda e o tamanho da fonte
ax.grid(True) # adiciona a grade

plt.show() # exibe o gráfico no ambiente de saída

3.2 Tipos Básicos de Gráficos#

Matplotlib suporta uma grande variedade de tipos de gráficos, e entre os mais comuns estão: gráficos de linha, gráficos de dispersão (scatter plots), gráficos de barras, histogramas, gráficos de pizza (pie charts) e box plots. Vamos explorar os mais comuns:

3.2.1 Gráfico de Linhas#

Um gráfico de linhas é uma representação visual de dados que exibe informações como uma série de pontos de dados conectados por segmentos de linha reta. Ele é frequentemente usado para mostrar tendências ao longo do tempo ou para comparar diferentes conjuntos de dados em um intervalo contínuo. Cada ponto no gráfico corresponde a um valor de dado, e a linha que os conecta ajuda a visualizar a progressão ou relação entre esses valores, tornando-o uma ferramenta eficaz para identificar padrões e mudanças em dados sequenciais.

# Dados
# Cria 100 pontos igualmente espaçados de 0 até 2π (um ciclo completo das funções trigonométricas)
x = np.linspace(0, 2*np.pi, 100)
y1 = np.sin(x) # calcula o seno para cada valor de x
y2 = np.cos(x) # calcula o cosseno para cada valor de x

# Criar gráfico
fig, ax = plt.subplots(figsize=(10, 6)) # define o tamanho da figura em polegadas
# Para o seno: define o rótulo da legenda, a linha da cor azul e a espessura da linha
ax.plot(x, y1, label='sen(x)', color='blue', linewidth=2)
# Para o cosseno: define o rótulo da legenda, a linha da cor vermelha, a espessura da linha e estilo tracejado
ax.plot(x, y2, label='cos(x)', color='red', linewidth=2, linestyle='--')

# Personalizar gráfico
ax.set_title('Funções Seno e Cosseno', fontsize=15) # define o título e o tamanho da fonte
ax.set_xlabel('x', fontsize=12) # define o rótulo do eixo x e o tamanho da fonte
ax.set_ylabel('y', fontsize=12) # define o rótulo do eixo y e o tamanho da fonte
ax.legend(fontsize=12) # adiciona a legenda e define o tamanho da fonte
ax.grid(True) # adiciona a grade
ax.set_ylim(-1.5, 1.5) # define os limites do eixo y

plt.show() # exibe o gráfico

Este script cria um gráfico de linhas que exibe, no mesmo plano cartesiano, as funções seno e cosseno ao longo de um ciclo completo, permitindo visualizar e comparar o comportamento dessas duas funções trigonométricas. As curvas são diferenciadas por cores e estilos de linha, com legendas, rótulos nos eixos, título e grade, tornando a análise das variações e relações entre seno e cosseno mais clara e intuitiva.

3.2.2 Gráfico de Dispersão (Scatter)#

Um gráfico de dispersão, também conhecido como scatter plot, é uma representação visual de dados que exibe a relação entre duas variáveis numéricas. Cada ponto no gráfico representa um par de valores, com a posição do ponto determinada pelos valores das variáveis nos eixos horizontal (x) e vertical (y). Gráficos de dispersão são utilizados para identificar padrões, tendências, agrupamentos ou correlações entre as variáveis, permitindo analisar se existe uma relação positiva, negativa ou nenhuma relação aparente entre os dados. Eles são especialmente úteis para visualizar a distribuição de dados e identificar outliers.

# Dados
np.random.seed(1) # semente para garantir reprodutibilidade
x = np.random.rand(50) # cria um array com 50 números aleatórios entre 0 e 1 para o eixo x
y = np.random.rand(50) # cria um array com 50 números aleatórios entre 0 e 1 para o eixo y
colors = np.random.rand(50) # um array com 50 números aleatórios entre 0 e 1, que serão usados para definir as cores de cada ponto no gráfico
sizes = 1000 * np.random.rand(50) # cria um array com 50 números aleatórios entre 0 e 1, multiplica cada número por 1000, e usa esses valores para definir o tamanho de cada ponto no gráfico

# Criar gráfico
fig, ax = plt.subplots(figsize=(10, 6))
# Criar o gráfico de dispersão, definindo o conjunto de dados, as cores e os tamanhos de cada ponto e a transparência
# cmap especifica uma paleta de cores, no caso 'viridis', que varia suavemente do roxo ao amarelo
scatter = ax.scatter(x, y, c=colors, s=sizes, alpha=0.6, cmap='viridis')

# Personalizar gráfico
ax.set_title('Gráfico de Dispersão', fontsize=15)
ax.set_xlabel('Eixo X', fontsize=12)
ax.set_ylabel('Eixo Y', fontsize=12)
ax.grid(True)

# Adicionar barra de cores
cbar = plt.colorbar(scatter) # adiciona uma barra de cores ao gráfico
cbar.set_label('Valores de Cor', fontsize=12) # define o rótulo da barra de cores

plt.show()

O script cria um gráfico de dispersão onde a posição de cada ponto é determinada por valores aleatórios em x e y, a cor de cada ponto é determinada por um valor aleatório mapeado pelo colormap viridis, e o tamanho de cada ponto também é determinado por um valor aleatório. A barra de cores (cbar) é adicionada para ajudar a interpretar a relação entre os valores numéricos e as cores no gráfico.

3.2.3 Gráfico de Barras#

Um gráfico de barras é uma representação visual de dados categóricos, onde cada categoria é representada por uma barra retangular. O comprimento ou altura de cada barra é proporcional ao valor que ela representa, permitindo uma comparação fácil e rápida entre diferentes categorias. Os gráficos de barras são frequentemente usados para exibir e comparar quantidades, frequências ou outras medidas estatísticas.

# Dados
categorias = ['A', 'B', 'C', 'D', 'E']
valores = [25, 40, 30, 55, 15]

# Criar gráfico
fig, ax = plt.subplots(figsize=(10, 6))
# Criar o gráfico de barras, definindo o conjunto de dados, a cor do preenchimento das barras e a cor da borda das barras
bars = ax.bar(categorias, valores, color='skyblue', edgecolor='navy')

# Personalizar gráfico
ax.set_title('Gráfico de Barras', fontsize=15)
ax.set_xlabel('Categorias', fontsize=12)
ax.set_ylabel('Valores', fontsize=12)
ax.grid(True, axis='y', linestyle='--', alpha=0.7) # adiciona a grade apenas de linhas horizontais, com estilo tracejado e transparência

# Adicionar valores acima das barras
for bar in bars: # loop que perrcorre cada barra
    height = bar.get_height() # obtém a altura da barra
    ax.text(bar.get_x() + bar.get_width()/2., height + 0.4, # calcula a posição X do texto, centralizando-o e o posiciona um pouco acima da altura da barra
            f'{height}', ha='center', va='bottom', fontsize=11) # define o texto a ser exibido, alinha horzontal e vertical e o tamanho da fonte

plt.show()

Este script gera um gráfico de barras que representa os valores associados a diferentes categorias, exibindo cada categoria no eixo horizontal e seu respectivo valor no eixo vertical, com os valores numéricos mostrados acima de cada barra para facilitar a leitura e comparação direta entre as categorias.

3.2.4 Histograma#

Um histograma é uma representação gráfica da distribuição de frequência de dados numéricos contínuos. Ele divide os dados em intervalos (bins) e exibe a frequência (ou contagem) de valores que caem em cada intervalo como barras adjacentes. A área de cada barra é proporcional à frequência dos valores no intervalo correspondente, permitindo visualizar a forma da distribuição, identificar tendências centrais, dispersão e a presença de outliers nos dados.

# Dados
np.random.seed(1) # Define uma semente para reprodutibilidade
data = np.random.normal(0, 1, 1000)  # Gera 1000 números aleatórios com distribuição normal

# Criar gráfico
fig, ax = plt.subplots(figsize=(10, 6))
# Plotar um histograma com 30 intervalos, definindo a cor das barras, das bordas e a transparência
n, bins, patches = ax.hist(data, bins=30, color='skyblue', edgecolor='black', alpha=0.7)

# Personalizar gráfico
ax.set_title('Histograma', fontsize=15)
ax.set_xlabel('Valores', fontsize=12)
ax.set_ylabel('Frequência', fontsize=12)
ax.grid(True, axis='y', linestyle='--', alpha=0.7)

# Adicionar linha de densidade
x = np.linspace(-4, 4, 100) # Cria 100 pontos entre -4 e 4
y = stats.norm.pdf(x, 0, 1) * len(data) * (bins[1] - bins[0]) # calcula a função de denidade de probabilidade normal e ajuste a escala da curva para corresponder ao histograma
ax.plot(x, y, 'r-', linewidth=2, label='Distribuição Normal') # plota a curva, definindo a cor e a espessura da linha e adiciona uma legenda
ax.legend(fontsize=12)

plt.show() # exibe o gráfico

Este script cria um histograma que mostra a distribuição dos dados gerados aleatoriamente, sobreposto por uma curva de densidade normal teórica, permitindo comparar visualmente se os dados seguem uma distribuição normal.

3.2.5 Gráfico de Pizza#

Um gráfico de pizza, também conhecido como gráfico de setores, é uma representação visual de dados em formato circular, onde cada fatia do círculo corresponde a uma categoria e seu tamanho é proporcional à quantidade ou porcentagem que essa categoria representa em relação ao total. Esse tipo de gráfico é especialmente útil para mostrar a participação relativa de cada parte em um todo, facilitando a comparação visual entre as proporções das diferentes categorias.

# Dados
categorias = ['A', 'B', 'C', 'D', 'E']
valores = [25, 40, 30, 55, 15]

# Criar gráfico
fig, ax = plt.subplots(figsize=(10, 6))
# Criar o gráfico de pizza, definindo o conjunto de dados, rótulas para cada fatia e formata os valores percentuais em uma casa decimal
wedges, texts, autotexts = ax.pie(valores, labels=categorias, autopct='%1.1f%%',
                                  startangle=90, shadow=True, # inicia a primeira fatia a partir do ângulode 90 graus e adiciona uma sombra ao gráfico
                                  wedgeprops={'edgecolor': 'black'}) # define a cor da borda de cada fatia

# Personalizar gráfico
ax.set_title('Gráfico de Pizza', fontsize=15) # define o título e o tamanho da fonte
plt.setp(autotexts, size=12, weight='bold') # define o tamanho da fonte e o estilo como negrito para os textos percentuais
ax.axis('equal')  # Garante que o gráfico seja circular

plt.show() # exibe o gráfico

O código cria um gráfico de pizza com fatias proporcionais aos valores fornecidos, rótulos para cada fatia, valores percentuais formatados, uma sombra para dar profundidade, bordas pretas nas fatias e garante que o gráfico seja circular. O título e o estilo dos textos percentuais também são personalizados.

3.2.6 Boxplot#

Um boxplot, também conhecido como diagrama de caixa, é uma representação gráfica que resume visualmente a distribuição de um conjunto de dados através de quartis. Ele exibe a mediana, quartis (Q1 e Q3), valores mínimos e máximos, além de identificar possíveis valores atípicos (outliers). Esta visualização é particularmente útil para analisar a dispersão, simetria e presença de valores discrepantes em um conjunto de dados.

# Gerar dados aleatórios para demonstração
np.random.seed(42)  # Define uma semente para reprodutibilidade
dados = np.random.normal(100, 15, 200)  # Gera 200 pontos com distribuição normal

# Criar figura e eixos
fig, ax = plt.subplots(figsize=(10, 6))  # Define o tamanho da figura

# Criar o boxplot com personalização
bp = ax.boxplot(dados,  # Dados a serem plotados
                patch_artist=True,  # Permite preenchimento das caixas
                medianprops={'color': 'black', 'linewidth': 2},  # Estilo da linha da mediana
                flierprops={'marker': 'o', 'markerfacecolor': 'gray'},  # Estilo dos outliers
                boxprops={'facecolor': 'lightblue', 'color': 'black'},  # Cor da caixa e borda
                whiskerprops={'color': 'black'},  # Cor dos whiskers
                capprops={'color': 'black'})  # Cor das linhas horizontais dos whiskers

# Personalizar o gráfico
ax.set_title('Boxplot', fontsize=15)  # Adiciona título
ax.set_ylabel('Valores', fontsize=12)  # Rótulo do eixo Y
ax.grid(True, axis='y', linestyle='--', alpha=0.7)  # Adiciona grade horizontal

plt.show()  # Exibe o gráfico

Este script gera um boxplot que representa visualmente a distribuição dos dados através de uma caixa central que contém o intervalo interquartil (IQR), uma linha horizontal destacada indicando a mediana, whiskers que se estendem até os valores mínimos e máximos não discrepantes, e pontos individuais que representam os outliers. O gráfico utiliza um esquema de cores com preenchimento azul claro para a caixa e elementos em preto para melhor contraste e visualização.

3.2.7 Subplots: Múltiplos Gráficos#

Matplotlib permite criar múltiplos gráficos em uma única figura:

# Dados
x = np.linspace(0, 10, 100)
y1 = np.sin(x)
y2 = np.cos(x)
y3 = np.tan(x)
y4 = x**2

# Criar figura com subplots, com uma matriz 2 x 2, duas linhas e duas colunas
fig, axs = plt.subplots(2, 2, figsize=(12, 10))

# Gráfico 1: Seno
axs[0, 0].plot(x, y1, 'b-') # Subplot superior esquerdo
axs[0, 0].set_title('Função Seno')
axs[0, 0].set_ylim(-1.5, 1.5)
axs[0, 0].grid(True)

# Gráfico 2: Cosseno
axs[0, 1].plot(x, y2, 'r-') # Subplot superior direito
axs[0, 1].set_title('Função Cosseno')
axs[0, 1].set_ylim(-1.5, 1.5)
axs[0, 1].grid(True)

# Gráfico 3: Tangente
axs[1, 0].plot(x, y3, 'g-') # Subplot inferior esquerdo
axs[1, 0].set_title('Função Tangente')
axs[1, 0].set_ylim(-5, 5)
axs[1, 0].grid(True)

# Gráfico 4: Quadrática
axs[1, 1].plot(x, y4, 'y-') # Subplot inferior direito
axs[1, 1].set_title('Função Quadrática')
axs[1, 1].grid(True)

# Visualização da matriz:
# [0,0] [0,1]
# [1,0] [1,1]

# Ajustar layout
plt.tight_layout() # ajusta automaticamente o espaçamento e a distribuição entre os suplots
plt.show()

3.2.8 Personalização de Gráficos#

Matplotlib oferece amplas opções de personalização:

# Dados
x = np.linspace(0, 10, 100)
y = np.sin(x)

# Criar gráfico com estilo personalizado
plt.figure(figsize=(12, 6))

# Definir estilo
plt.style.use('seaborn-v0_8-darkgrid') # aplica um estilo predefinido do Seaborn (darkgrid) para melhorar a estética do gráfico

# Plotar a função seno com uma linha vermelha sólida ('r-') de espessura 2, e adiciona uma legenda com o rótulo 'sen(x)'
plt.plot(x, y, 'r-', linewidth=2, label='sen(x)')

# Personalizar título e rótulos
# Definir o título do gráfico com tamanho de fonte 20, em negrito, e adiciona um espaçamento de 20 pontos entre o título e o gráfico.
plt.title('Função Seno Personalizada', fontsize=20, fontweight='bold', pad=20)
# Definir os rótulos dos eixos X e Y com tamanho de fonte 14, em negrito, e adiciona um espaçamento de 10 pontos entre os rótulos e os eixos.
plt.xlabel('Eixo X', fontsize=14, fontweight='bold', labelpad=10)
plt.ylabel('Eixo Y', fontsize=14, fontweight='bold', labelpad=10)

# Personalizar eixos
# Definir o tamanho da fonte dos valores nos eixos X e Y como 12.
plt.xticks(fontsize=12)
plt.yticks(fontsize=12)
# Adicionar uma grade ao gráfico com linhas tracejadas e transparência de 70%.
plt.grid(True, linestyle='--', alpha=0.7)

# Adicionar anotações
plt.annotate('Pico', xy=(np.pi/2, 1), xytext=(np.pi/2 + 1, 0.8), # coordenadas do ponto a ser anotado e do texto da anotação
             arrowprops=dict(facecolor='black', shrink=0.05, width=1.5), # propriedades da seta, cor, encolhimento e largura
             fontsize=12) # define o tamanho da fonte como 12

# Adicionar legenda ao gráfico com tamanho de fonte 14, localizada no canto inferior direito
plt.legend(fontsize=14, loc='lower right')

# Adicionar texto ao gráfico na posição (7, 0.5) exibindo a expressão matemática "sin(x)" com tamanho de fonte 20.
plt.text(7, 0.5, r'$\sin(x)$', fontsize=20)

# Adicionar linhas de referência (cor preta, estilo sólida e transparência de 30%)
plt.axhline(y=0, color='k', linestyle='-', alpha=0.3) # linha horizontal no eixo y = 0
plt.axvline(x=np.pi, color='k', linestyle='-', alpha=0.3) # linha vertical no eixo x = pi

# Definir os limites dos eixos X e Y para melhorar a visualização do gráfico
plt.xlim(0, 10)
plt.ylim(-1.5, 1.5)

plt.tight_layout() # ajuste automático do espaçamento dos elementos do gráfico
plt.show() # exibe o gráfico

O código cria um gráfico da função seno com diversas personalizações estéticas e informativas, incluindo estilo visual, título, rótulos, grade, anotações, legenda, texto e linhas de referência, resultando em uma visualização clara e detalhada da função.

# Criação dos dados: array de anos de 2015 a 2024 e listas com dados populacionais fictícios
anos = np.arange(2015, 2025)  # Cria um array com anos consecutivos
cidade_a = [120, 125, 130, 138, 145, 150, 158, 165, 172, 180]  # População da cidade A
cidade_b = [80, 85, 90, 92, 95, 100, 105, 110, 115, 120]      # População da cidade B
cidade_c = [60, 62, 65, 70, 75, 80, 85, 90, 95, 100]          # População da cidade C

# Define as cores para cada linha do gráfico usando códigos hexadecimais
cores = ['#1f77b4',  # Azul
         '#ff7f0e',  # Laranja
         '#2ca02c']  # Verde

# Cria uma nova figura com tamanho 10x6 polegadas
fig, ax = plt.subplots(figsize=(10, 6))

# Define o estilo visual do gráfico como seaborn-white (fundo branco com elementos do seaborn)
plt.style.use('seaborn-v0_8-white')

# Plota as três linhas do gráfico, cada uma com sua cor e espessura
ax.plot(anos, cidade_a, color=cores[0], linewidth=2)  # Linha cidade A
ax.plot(anos, cidade_b, color=cores[1], linewidth=2)  # Linha cidade B
ax.plot(anos, cidade_c, color=cores[2], linewidth=2)  # Linha cidade C

# Adiciona rótulos no final de cada linha, usando a mesma cor da linha correspondente
ax.text(anos[-1]+0.2, cidade_a[-1], 'Cidade A', color=cores[0], va='center', fontsize=12)  # Rótulo cidade A
ax.text(anos[-1]+0.2, cidade_b[-1], 'Cidade B', color=cores[1], va='center', fontsize=12)  # Rótulo cidade B
ax.text(anos[-1]+0.2, cidade_c[-1], 'Cidade C', color=cores[2], va='center', fontsize=12)  # Rótulo cidade C

# Remove as bordas superior e direita do gráfico para um visual mais limpo
ax.spines['top'].set_visible(False)    # Remove borda superior
ax.spines['right'].set_visible(False)  # Remove borda direita

# Ajusta a espessura das bordas esquerda e inferior
ax.spines['left'].set_linewidth(1.5)    # Espessura da borda esquerda
ax.spines['bottom'].set_linewidth(1.5)  # Espessura da borda inferior

# Ajusta os limites do eixo X para acomodar os rótulos das linhas
ax.set_xlim(anos[0], anos[-1]+1)

# Define os rótulos dos eixos X e Y com tamanho de fonte específico
ax.set_xlabel('Ano', fontsize=13)
ax.set_ylabel('População (milhares)', fontsize=13)

# Define o título do gráfico
ax.set_title('Evolução da População nas Cidades', fontsize=15)

# Adiciona grade horizontal (eixo y) com linha tracejada e 70% de opacidade
ax.grid(True, axis='y', linestyle='--', alpha=0.7)

# Ajusta automaticamente o layout para evitar sobreposição de elementos
plt.tight_layout()

# Exibe o gráfico
plt.show()

Esse gráfico mostra a evolução populacional de três cidades fictícias ao longo de uma década, com as linhas identificadas ao final de cada uma, facilitando a leitura e comparação direta.

3.2.9 Salvando Gráficos#

Matplotlib permite salvar gráficos em vários formatos. Formatos de Arquivo:

a) PNG (Portable Network Graphics)

  • Formato com compressão sem perda

  • Ideal para imagens com texto, linhas nítidas e áreas de cor sólida

  • Suporta transparência

  • Melhor escolha para uso web e apresentações

b) PDF (Portable Document Format)

  • Formato vetorial

  • Mantém a qualidade em qualquer escala

  • Ideal para documentos acadêmicos e publicações

  • Tamanho de arquivo geralmente menor que PNG em alta resolução

c) SVG (Scalable Vector Graphics)

  • Formato vetorial baseado em XML

  • Pode ser editado posteriormente

  • Mantém a qualidade em qualquer escala

  • Ideal para web e edição posterior

d) JPG/JPEG

  • Formato com compressão com perda

  • Menor tamanho de arquivo

  • Melhor para imagens fotográficas

  • Não recomendado para gráficos com texto ou linhas finas

DPI, ou Dots Per Inch (pontos por polegada), é uma medida de resolução que indica a densidade de pontos individuais (pixels) em uma polegada de uma imagem ou impressão. Quanto maior o DPI, mais pontos são usados para representar a imagem, resultando em maior detalhe e nitidez. Em contextos de impressão, um DPI mais alto geralmente significa uma qualidade superior, enquanto para visualização em tela, um DPI menor pode ser suficiente, dependendo do tamanho da tela e da distância de visualização.

  • 300 DPI é considerado padrão para impressão de qualidade

  • Para web, 72-150 DPI geralmente é suficiente

  • Não afeta formatos vetoriais (PDF, SVG)

O parâmetro bbox_inches='tight' na função savefig do Matplotlib é utilizado para ajustar automaticamente as bordas da figura, removendo espaços em branco desnecessários ao redor do gráfico. Ao especificar bbox_inches='tight', o Matplotlib calcula o menor bounding box que contém todos os elementos do gráfico (como linhas, texto, legendas, etc.) e salva a figura com essas bordas ajustadas, garantindo que nenhum detalhe seja cortado e otimizando o uso do espaço disponível na imagem resultante.

# Criar um gráfico de caminhada aleatória
plt.figure(figsize=(10, 6))

# Configurar a semente aleatória para reprodutibilidade
np.random.seed(1)

# Gerar e plotar dados de caminhada aleatória
dados_aleatorios = np.random.randn(50).cumsum()
plt.plot(dados_aleatorios, 'r-', linewidth=2, label='Variação Acumulada')

# Adicionar título e rótulos dos eixos
plt.title('Caminhada Aleatória - Variação Acumulada', fontsize=14)
plt.xlabel('Número de Passos', fontsize=12)
plt.ylabel('Valor Acumulado', fontsize=12)

# Adicionar grade
plt.grid(True, linestyle='--', alpha=0.7)

# Adicionar legenda
plt.legend(fontsize=10)

# Ajustar layout
plt.tight_layout()

# Exibir gráfico
plt.show()

# Salvar em diferentes formatos
plt.savefig('grafico.png', dpi=300, bbox_inches='tight')  # PNG
plt.savefig('grafico.pdf', bbox_inches='tight')  # PDF
plt.savefig('grafico.svg', bbox_inches='tight')  # SVG
plt.savefig('grafico.jpg', dpi=300, bbox_inches='tight')  # JPG

plt.show()
